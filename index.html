🎯 Format Americano

Le tournoi Americano est un format convivial où les partenaires changent à chaque match. Chaque joueur accumule ses points individuellement.

Rôles

  • 👑 Admin : configure le tournoi, gère les joueurs, lance et administre.
  • 🎯 Opérateur : saisit les scores et soumet des demandes de substitution à l'admin.
  • 👁 Spectateur : consulte les matchs et le classement en temps réel.

Points

  • Victoire : 3 pts · Nul : 2 pts · Défaite : 0 pt
